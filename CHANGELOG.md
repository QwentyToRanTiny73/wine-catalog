# CHANGELOG

## [feature/table-update-from-excel] — 2025-04-05

### Добавлено

- **`data/analytical_data.xlsx`** — источник данных, файл «БД_133_образца_исправлено.xlsx»
  (133 образца виноматериалов Крыма, урожаи 2017–2025).

- **`data/table.js`** — клиентский JavaScript-модуль:
  - Загрузка Excel-файла с сервера через `fetch()` + SheetJS (`xlsx`).
  - Автоматическое определение строки заголовков.
  - Парсинг всех 37 колонок с маппингом в JS-поля.
  - Постраничный рендеринг таблицы (30 строк / страница) — не грузит DOM сразу.
  - Глобальный поиск + 3 фильтра (год, цвет, регион).
  - Сортировка по любому столбцу (клик по заголовку).
  - Кнопка **«Доп.»** → модальное окно с полным аналитическим досье образца.
  - Закрытие модала: крестик / клик снаружи / `Esc`.

- **`data/table.css`** — стили таблицы и модального окна:
  - Фиксированная шапка таблицы (`position: sticky`).
  - Горизонтальная прокрутка таблицы на узких экранах.
  - Цветные бейджи для типа вина (белое / красное / розовое).
  - Адаптивность от 320 px: на мобильных кнопка «Доп.» трансформируется в иконку ℹ️.
  - Анимации модального окна (fade + slide-up).

### Изменено

- **`index.html`** — главная страница:
  - Добавлена секция с таблицей (`#table-section`).
  - Подключены `data/table.css` и `data/table.js` (через `defer`).
  - SheetJS подключён через CDN (`defer`).
  - Модальное окно вынесено в конец `<body>` для правильного z-index.

- **`.github/workflows/static.yml`** — исправлен триггер деплоя:
  - Изменена ветка деплоя с `qwen-code-93ce8202-...` на **`main`**.
  - Теперь GitHub Pages автоматически обновляется при пуше в `main`.

### Структура аналитического блока в модальном окне

1. **Организационный блок** — район, зона, населённый пункт, производитель, дата.
2. **Показатели по ГОСТ** (регламентируемые):
   - Объёмная доля этил. спирта (% об.)
   - Приведённый экстракт (г/дм³)
   - Титруемая кислотность (г/дм³)
   - Сахара (г/дм³)
3. **Дополнительные показатели** — 19 показателей: органические кислоты (лимонная,
   винная, яблочная, янтарная, молочная, уксусная), сахара (глюкоза, фруктоза,
   биозы), глицерин, минеральный состав (хлориды, Na, K, Mg, Ca), фенольные
   вещества, pH, буферная ёмкость, электропроводность.
