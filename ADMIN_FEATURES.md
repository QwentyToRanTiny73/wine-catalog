# Система управления винными образцами

## Описание функционала

### Обычный пользователь
- Может просматривать каталог образцов вина
- Использовать фильтры и сортировки
- Просматривать детальную информацию об образцах
- Экспортировать данные в Excel и Word
- Переходить между страницами каталога

### Администратор
- Имеет доступ к личному кабинету через форму входа
- Может добавлять новые образцы вина
- Видит статистику по винам
- Может выйти из системы

## Структура файлов

- `index.html` - Главная страница с доступом к каталогу и личному кабинету
- `catalog.html` - Каталог вин с фильтрами, сортировками и пагинацией
- `admin_login.html` - Страница входа для администраторов
- `admin_panel.html` - Личный кабинет администратора с возможностью добавления образцов
- `js/data.js` - Файл с начальными данными о винах
- `js/script.js` - Основной скрипт для каталога вин

## Авторизация

- Система использует localStorage для хранения сессии администратора
- При входе в систему, значение `isAdminLoggedIn` устанавливается в 'true'
- При выходе из системы значение удаляется
- Только при наличии этой сессии отображаются элементы управления для администратора

## Учетные данные

- Логин: `admin`
- Пароль: `admin123`

## Особенности реализации

- Все добавленные данные сохраняются в localStorage браузера
- При перезагрузке страницы данные восстанавливаются из localStorage
- Если данных в localStorage нет, используется оригинальный набор из js/data.js
- При добавлении новых образцов ID генерируется автоматически на основе максимального существующего ID
- Статистика в админ-панели обновляется в реальном времени
